# JqueryLista
 W tej liście jest dodawanie elementów od początku i końca, usuwanie elementów od początku i końca oraz cała liste, przywracanie jednego produktu oraz całej listy, sortowanie alfabeczyne, parzyste kolorowanie, czyszczenie kolorów oraz wyszukiwanie elementów z listy. Zadbałem o spójności logiczne oraz o wygląd i strukturę strony.
